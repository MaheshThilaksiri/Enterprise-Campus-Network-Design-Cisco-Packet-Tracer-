# Enterprise Campus Network Design

A secure enterprise campus network designed using Cisco Packet Tracer.

## Features
- VLAN Segmentation
- Inter-VLAN Routing
- DHCP Server
- DNS Server
- HTTP/HTTPS Server
- FTP Server
- Wireless Network
- ACL Security
- Syslog Monitoring
- Port Security
- HSRP Redundancy
- OSPF Routing
- SSH Remote Management
- Branch Office Connectivity

## Technologies
- Cisco Packet Tracer
- VLANs
- OSPF
- HSRP
- DHCP
- DNS
- FTP
- HTTP/HTTPS
- SSH
- ACLs
- Syslog
- Wireless Networking
